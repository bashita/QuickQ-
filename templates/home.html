How To Use

QuickQ is designed to be fast, simple, and accessible for everyone without requiring accounts or downloads.

To Create a Queue: Go to "Create Queue" tab, enter a name for your queue, and submit. You'll receive a unique queue code, admin token and a link to manage your queue.
To Join a Queue: Go to "Join Queue" tab, enter the queue code provided by the admin, and your nickname. You can also scan a QR code if available. You can check the queue status at any time with queue code.
To Manage a Queue: Go to "Manage Queue" tab, enter your admin token to access the queue management interface through the admin token, where you can see the queue status, call the next person of the queue.
